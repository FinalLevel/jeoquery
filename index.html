GeoNames +             

jeoQuery by Thomas Haukland

        
    
    

jeoQuery is a small jquery wrapper for the JSON methods offered by geonames. It is written and maintained as a hobby project by Thomas Haukland.

Methods demo The "jeoquery" namespace contains one function per geonames service, allowing you to call them easily(ok it isn't that hard without this wrapper, but this gives you autocomplete on params, and this really nice demo page where I've used jstree to visualize the resulting json.

UI demo In addition, the jeoquery.js file gives you some ready ui-plugins. Check out the ui-demo for a demonstration of the country picker and the postal code lookup. More to come soon...
